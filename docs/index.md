# Discord Sound Test — Policies

This small site hosts the canonical Privacy Policy and Terms of Use for the
developer-hosted instance of the Discord Sound Test bot. It is intended for
use with GitHub Pages (serve the repository's `docs/` folder).

- [Privacy Policy](privacy.md)
- [Terms of Use](terms.md)
- [License](../LICENSE)

To enable GitHub Pages for this repository:

1. Go to your repository on GitHub.
2. Settings → Pages.
3. Under "Source", select the `main` branch and the `/docs` folder.
4. Save. GitHub will publish the site within a few minutes.

Example URL once published:

```
https://deathmarcher.github.io/discord-sound-test/
```
